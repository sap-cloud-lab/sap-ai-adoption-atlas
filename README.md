# SAP AI Adoption Atlas

A static GitHub Pages site for SAP AI adoption conversations across:

- Embedded SAP application AI
- Joule and Joule Agents
- SAP BTP AI Foundation
- Finance close and reporting AI
- Public Cloud and Private Cloud adoption readiness
- Release-watch tracking for 2608 and future releases

The site is intentionally plain static HTML, CSS, and JavaScript so it can be hosted directly from GitHub Pages.

## Source notes

The capability catalogue is grounded in official SAP sources such as SAP Learning, SAP Help, SAP product pages, and SAP release learning pages. Release-watch content is deliberately separated from confirmed capability content until SAP publishes final release details.
